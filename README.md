## 📖 Sobre o Projeto

O projeto **We Care** consiste em uma landing page desenvolvida para praticar conceitos essenciais de desenvolvimento web, incluindo estruturação semântica, estilização avançada e responsividade.

Durante o desenvolvimento foram aplicados conceitos como:

* Estruturação semântica com HTML5
* Estilização utilizando CSS3
* Posicionamento de elementos
* Layout Desktop e Mobile
* Responsividade com Media Queries
* Tipografia personalizada com Google Fonts
* Organização de conteúdo visual e textual

---

## 🖥️ Layout Desktop

* Menu de navegação completo
* Botão de Login destacado
* Imagem ilustrativa ao lado do conteúdo
* Títulos com destaque visual
* Botão de chamada para ação (CTA)

---

## 📱 Layout Mobile

* Menu de navegação ocultado
* Conteúdo centralizado
* Imagem reposicionada para melhor visualização
* Texto simplificado para telas menores
* Layout otimizado para smartphones

## 🛠️ Como Executar o Projeto

1. Clone o repositório:

```bash

```

2. Acesse a pasta do projeto.

3. Abra o arquivo:

```text

```

4. O navegador carregará automaticamente os estilos definidos em:
